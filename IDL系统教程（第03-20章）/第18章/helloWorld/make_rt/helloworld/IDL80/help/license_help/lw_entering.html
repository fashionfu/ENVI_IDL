Entering a License

This screen allows you to select a license file on your system for the license wizard to import and save. Once a file is selected, the license information is saved in the appropriate location on your system.

The license wizard accepts files names <productname>.lic for evaluation licenses and license.dat for all other licenses.


Note
If you request a license via the web, the license will be saved automatically and the license wizard will advance past this screen.
